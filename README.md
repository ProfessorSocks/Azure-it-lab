# Microsoft Entra ID Lab

This repository documents my hands-on lab work with Microsoft Entra ID as part of my IT support training.

## Project Goal

The goal of this lab is to build practical familiarity with Microsoft identity and user administration concepts that are commonly used in entry-level IT support and help desk roles.

## Skills Practiced

- Navigating the Microsoft Entra admin center
- Understanding tenant and directory context
- Troubleshooting visibility issues in the admin portal
- Accessing the Users section in the correct tenant
- Investigating password reset limitations
- Thinking through identity-based troubleshooting steps

## Why This Lab Matters

Microsoft Entra ID is a core part of Microsoft 365 environments. Many common IT support issues begin with identity, authentication, user access, or tenant navigation. This lab helps build real-world familiarity with those workflows.

## Lab Topics

- Environment setup
- Tenant navigation and switching directories
- User management basics
- Password reset troubleshooting
- Lessons learned
- Next steps for continued training

## Notes

This lab is being documented step by step as if it were an internal IT training project. The goal is not just to click through the portal, but to understand what each action means and how to troubleshoot common problems.
