# User Management Basics

## Objective

The goal of this section is to understand how to locate, review, and manage users inside Microsoft Entra ID.

## Accessing Users

To view users:

1. Navigate to Microsoft Entra ID
2. Click **Users**
3. Select **All users**

This displays a list of all user accounts in the tenant.

## Key Observations

- The user list includes all accounts within the current directory
- If the correct tenant is not selected, users may not be visible
- Searching for users can be done using the search bar at the top

## User Properties

When selecting a user, several important attributes can be reviewed:

- Username (User Principal Name)
- Account status (enabled/disabled)
- Assigned roles
- Group memberships

## Common Tasks

Basic user-related tasks include:

- Viewing account details
- Verifying account status
- Checking assigned roles
- Reviewing group membership

## Troubleshooting Mindset

When dealing with user-related issues, the first step is always to verify:

1. The user exists
2. The correct account is being referenced
3. The account is not disabled or blocked

## Lessons Learned

Accessing users depends heavily on being in the correct tenant. If users are not visible, it does not necessarily mean they do not exist—it may indicate a context issue.

## IT Support Relevance

User management is a foundational task in IT support. Many issues begin with verifying the user account before moving on to deeper troubleshooting steps.
